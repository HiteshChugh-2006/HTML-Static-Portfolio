## ✨ Sections

- 👤 **About Me** — Introduction and personal summary
- 🛠️ **Skills** — Technical skills and competencies
- 💼 **Projects** — Showcase of work
- 📬 **Contact** — Links and contact information

## 🚀 Getting Started

```bash
git clone https://github.com/HiteshChugh-2006/HTML-Static-Portfolio.git
open index.html
```

## 🌐 Deploy to GitHub Pages

1. Go to **Settings → Pages**
2. Source: `main` branch → `/(root)`
3. Visit `https://HiteshChugh-2006.github.io/HTML-Static-Portfolio`
